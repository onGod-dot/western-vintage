# Western Vintage Organization

A modern website for Western Vintage Organization and the Western Vintage Gala 2025.

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
